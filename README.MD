# 🚀 freeCodeCamp JavaScript Algorithms and Data Structures

Ласкаво просимо до мого навчального репозиторію! Тут зібрані всі практичні завдання та проекти, які я виконав під час проходження курсу від **freeCodeCamp**. Це повний шлях від основ синтаксису до складних асинхронних додатків та сертифікаційних проектів.

## 🛠 Технологічний стек

- **Core:** JavaScript ES6+
- **Web API:** DOM Manipulation, localStorage, Fetch API
- **Concepts:** OOP (Classes), Functional Programming, Recursion, Regex
- **Tools:** HTML5, CSS3, Markdown

---

## 📂 Навігація по навчальному курсу

Усі завдання розділені за темами. Кожна папка містить код рішення (`script.js`) та опис завдання (`task.md`).

### 基礎 Основи (Modules 01 - 07)

- **01-Variables-and-Strings** — Робота зі змінними та рядками.
- **02-Booleans-and-Numbers** — Логічні типи та математика.
- **03-Functions** — Створення функцій (Email Masker, Leap Year Calculator).
- **04-Arrays** — Основи масивів (Lunch Picker, Golf Score).
- **05-Objects** — Об'єкти та структури даних (Quiz Game, Record Collection).
- **06-Loops** — Цикли та алгоритми (Factorial, Longest Word Finder).
- **07-JavaScript-Fundamentals-Review** — Великий огляд (Pyramid Generator, Password Generator).

### ⚡ Просунуті концепції (Modules 08 - 14)

- **08-Higher-Order-Functions** — Коллбеки та функції вищого порядку.
- **09-DOM-Manipulation-and-Events** — Робота з інтерфейсом (Lightbox, Character Counter).
- **10-JavaScript-and-Accessibility** — Доступність та семантика.
- **11-Debugging** — Пошук та виправлення помилок.
- **12-Basic-Regex** — Регулярні вирази (Palindrome Checker, Pig Latin).
- **13-Form-Validation** — Валідація форм.
- **14-Dates** — Робота з часом та форматами дат.

### 🧩 Складні структури та логіка (Modules 15 - 21)

- **15-Audio-and-Video-Events** — Робота з медіа.
- **16-Maps-and-Sets** — Сучасні колекції даних.
- **17-localStorage-and-CRUD** — Збереження даних у браузері.
- **18-Classes** — Об'єктно-орієнтоване програмування (Project Idea Board).
- **19-Recursion** — Глибоке занурення в рекурсію (Permutation Generator).
- **20-Functional-Programming** — Чисті функції та каррування (Optional Arguments Sum).
- **21-Asynchronous-JavaScript** — Робота з API (fCC Forum Leaderboard).
